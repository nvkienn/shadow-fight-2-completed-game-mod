# shadow fight 2 completed game mod

Note: Using the XML Hack requires an Android device. It CANNOT be done on an Apple device.

### Features

- Play the hardest bosses right away: Eclipse May/Lynx/Hermit etc.

- Infinite money/gem/energy

- All sets/mythical enchantments unlocked

- Raids are available

### Installation Instructions

IMPORTANT: If you want to save your current game state, you need to copy the users.xml file somewhere else and keep it safe so you can always load back your game state. The users.xml file can be found in Android>data>com.nekki.shadowfight>files>userdata. You may need to connect your phone to a computer to be able to access the file.

Step 1: Download `shadow-fight-2-2-8-0.apk` and `users.xml` from [releases](https://github.com/nvkienn/shadow-fight-2-completed-game-mod/releases).

Step 2: Uninstall the current Shadow Fight 2 on your device.

Step 3: Disable Internet connection, both Wi-fi and Celluar.

Step 4: Open the "My Files" app on your device.

Step 5: Locate the Downloads folder. Install the APK file provided, shadow-fight-2-2-8-0.apk.

Step 6: Launch the game and wait until you see Sensei's scorn greeting. Quit the game.

Step 7: You will likely need to connect your phone to a computer to continue with the file editing. Navigate to Android>data>com.nekki.shadowfight>files>userdata and delete users.xml and users_backup.xml. Then copy the users.xml provided here into the userdata folder.

Step 8: Enable your Internet and launch the game. After launching the game, you will be prompted to update the game through Google Play. Start the update.

Step 9: In the userdata folder, delete users.xml.hash. After you delete the file, rename the users_backup.xml.hash file to users.xml.hash.

You are done! Wait for the update to finish, then launch the game. You will be prompted to download about 500MB worth of files for the game, and when it's done, the game should already have been completed and have infinite money/gems and the rest of the features listed above!
